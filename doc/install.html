Installation

The installation procedure is similar to Javalib. Moreover, you must install Javalib before installing Sawja.

(optional:) In order to obtain better performaces for XTA class analysis, you should use BuDDy (a BDD package).
For that, you have to download BuDDy, apply the patch buddycaml/patch on source files and install it.
And finally use ./configure.sh with option -b yes.
